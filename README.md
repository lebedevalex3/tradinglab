# TradingLab

TradingLab — учебно-инженерный проект для освоения профессионального подхода
к разработке, тестированию и анализу торговых стратегий.

Цель проекта — не поиск «волшебной стратегии», а построение **воспроизводимого, аналитически корректного ресёрч-процесса**:
от идеи и спецификации до бектестов, диагностики и приближённого к реальности исполнения.

---

## Цели проекта

- Освоить профессиональный подход к тестированию стратегий
- Научиться работать с Python-бектестами и аналитикой
- Понимать механику индикаторов и режимов рынка
- Использовать Freqtrade как симулятор реального исполнения
- Строить стратегии как инженерные системы, а не наборы сигналов

---

## Структура проекта (кратко)

- `pine/` — индикаторы и стратегии TradingView (PineScript)
- `src/tradinglab/` — Python-библиотека (данные, бектесты, отчёты)
- `research/` — ноутбуки и аналитические отчёты
- `data/` — данные (sample-коммитятся, реальные — нет)
- `artifacts/` — результаты запусков (runs)
- `docs/` — спецификации и гайды
- `scripts/` — CLI-скрипты (точки входа)


## Результаты запуска

Каждый запуск создаёт папку:

```text
artifacts/runs/<run_id>/
```

Внутри:

- meta.json — паспорт прогона (версия, данные, параметры)
- summary.json — краткое резюме
- report.md — человекочитаемый отчёт
- trades.parquet — сделки (на Milestone 0 может быть пустым)
- config.yml — конфиг запуска
